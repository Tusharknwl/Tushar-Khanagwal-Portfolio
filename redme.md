## Setup and Usage
To run this portfolio locally, follow these steps:

1. Clone the repository:
git clone https://github.com/Tusharknwl/Tushar-Khanagwal-Portfolio.git


2. Navigate to the project directory:

`cd Tushar-Khanagwal-Portfolio`


3. Install the dependencies using a package manager like npm or yarn:
`npm install`


4. Create a .env file in the root directory and add the following variables:
`VITE_APP_EMAILJS_PUBLIC_KEY=your_emailjs_public_key`
`VITE_APP_EMAILJS_SERVICE_ID=your_emailjs_service_id`
`VITE_APP_EMAILJS_TEMPLATE_ID=your_emailjs_template_id`

You can obtain these values by following these steps:

- Go to EmailJS and create an account.
- Once logged in, create a new service and note down the Service ID.
- Create a new email template within the service and note down the Template ID.
- Go to the Service Settings tab, and under "API Keys," you'll  find your Public API Key. This key is your VITE_APP_EMAILJS_PUBLIC_KEY.



5. Start the development server:
`npm run dev`


6. Open your web browser and visit http://localhost:3000 or http://localhost:5173 to view the portfolio locally.

That's it! You should now be able to run the portfolio on your machine.
